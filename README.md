# BarberiaJK

Este proyecto corresponde a mi trabajo para la asignatura, donde desarrollé una API siguiendo la estructura en capas: Domain, Infrastructure y API. El sistema está basado en una barbería, e incluye entidades como Clientes, Empleados, Servicios, Citas y Comisiones. También agregué la carpeta db, donde incluí el script completo para crear la base de datos, y unos inserts de prueba para que todo se pueda ver funcionando de inmediato. Todo el código está organizado según los criterios solicitados, y la API permite hacer operaciones CRUD sobre cada entidad. Mi objetivo fue mantenerlo claro, funcional y fácil de probar.
